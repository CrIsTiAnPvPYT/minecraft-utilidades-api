# Minecraft Utilidades

_El npm que te ayudará a crear comandos relacionados con minecraft_

## Instalación 🚀

```
npm i minecraft-utilidades --save
```

### Constructores 🔧

_Lista de constructores_

```
    cabeza
    avatar
    cuerpo
    skin
    server
    server_img
    perfil
```

### Necesitan async? 🔧

```
    cabeza: No
    avatar: No
    cuerpo: No
    skin: No
    server: Si
    server_img: No
    perfil: Si
```
